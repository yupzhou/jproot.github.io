# JPRoot "JaPanese Root project." 日语音训查询 

by Yupzhou 
feat. EdgeLight

## 包含内容

- 以词根为中心的派生词汇表
- 列出各词缀的功能
- 将日本字音按《广韵》分类
- 日语简史

## 特色

- Live2d 话痨
- 因为 Live2d 的卡顿开设了 Lite 版（轻量版）
- 专为宽屏、手机设计页面
- 平板·手机推荐 Lite 版，电脑（或有 5G 或以上信号的移动端）推荐桌面版

## 查询功能

- 首页：根据搜索方式支持平假名/片假名/汉字
- 词根页、词汇页：支持平假名/片假名
- 音读页：支持汉字
- 模糊查询
- 支持平假名、片假名转换
