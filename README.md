# JPRoot "JaPanese Root project." 日语音训查询 

by Yupzhou feat. EdgeLight

## 包含内容

- 以词根为中心的派生词汇表
- 列出各词缀的功能
- 将日本汉字音按《广韵》分类
- 日语简史

## 查询功能

- 支持平假名/片假名
- 支持汉字
